# PIB - Personal Interview Brain

打破"看题就懂，说题就懵"的怪圈。

## 1. 核心愿景 (Vision)

通过 AI 语义比对与逻辑纠偏，配合遗忘曲线，在终端环境下（TUI）低调且高效地构建系统化的面试知识库。

## 2. 目标用户 (User Persona)

- **身份**: 准备跳槽或巩固技术的后端开发者
- **痛点**:
  - 看面经容易走神，缺乏交互感
  - 表达能力弱，面试时逻辑混乱
  - 在公司看网页面经太显眼，容易被老板发现
  - 碎片化知识多，难以形成长效记忆

## 3. 功能模块

### 3.1 题库管理 (Quest-Library)
- 导入/导出：支持 Markdown 批量导入，或从 Obsidian 笔记自动同步
- 标签系统：支持多维度标签（如：Golang, K8s, System-Design）
- AI Scout：给出一个题目，Agent 自动通过网络检索/生成 3 种维度的参考答案（入门级、专家级，大厂面试官版）

### 3.2 模拟练习 (The Mock-Coach)
- 浸入式回答：题目弹出后，进入纯文本输入模式
- AI 比对引擎：
  - 语义得分：计算用户输入与标准答案的向量距离
  - 表达分析：识别逻辑断层、废话过多或专业术语缺失
  - 追问模式：AI 会根据你的回答漏洞，随机生成 1-2 个 Follow-up Question
- 打字机监控：记录打字频率和停顿，分析哪些知识点是用户"犹豫"的

### 3.3 遗忘曲线 (Memo-Engine)
- 智能调度：采用 SM-2 算法，根据每次练习的 AI 评分自动排期下一次复习
- 复习优先级：每天打开 TUI，首屏自动展示"今日到期"和"薄弱环节"题目

### 3.4 可视化分析 (Analytics)
- 能力雷达图：实时展示 ASCII 字符构成的六边形图
- 进度追踪：展示从"New"到"Mastered"的题目状态迁移

### 3.5 Boss Mode (Stealth)
- 一键伪装：Esc 或自定义快捷键，瞬间切换到 tail -f /var/log/nginx/access.log 的伪造日志流界面
- 进程隐藏：在进程列表中显示为常见的系统服务名

## 4. 技术栈

| 维度 | 选型 | 理由 |
|------|------|------|
| 编程语言 | Golang | 单文件分发极其方便 |
| 存储层 | SQLite | 零配置，单文件隐藏性强 |
| TUI 框架 | Bubble Tea (Charm) | 交互体验最强，UI 组件丰富 |
| AI 集成 | OpenAI/DeepSeek API | 支持用户配置自己的 API Key |
| 部署方式 | Binary File | 一个可执行程序 + 一个 .db 隐藏文件 |
