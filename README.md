# PIB - Personal Interview Brain

打破"看题就懂，说题就懵"的怪圈。

## 1. 核心愿景 (Vision)

通过 AI 语义比对与逻辑纠偏，配合遗忘曲线，在终端环境下（TUI）低调且高效地构建系统化的面试知识库。

## 2. 目标用户 (User Persona)

- **身份**: 准备跳槽或巩固技术的后端开发者
- **痛点**:
  - 看面经容易走神，缺乏交互感
  - 表达能力弱，面试时逻辑混乱
  - 在公司看网页面经太显眼，容易被老板发现
  - 碎片化知识多，难以形成长效记忆

## 3. 产品特点

- 🎯 **TUI 终端界面**: 低调高效，在公司使用不显眼
- 🧠 **AI 辅助**: 语义比对 + 逻辑纠偏
- 📅 **遗忘曲线**: 基于 SM-2 算法智能安排复习
- 💾 **本地存储**: SQLite，数据安全可控

